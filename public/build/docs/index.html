prokodo CLI

The prokodo CLI is the official command-line interface for the prokodo marketplace. It lets you authenticate, configure projects, upload files for cloud verification, and inspect your environment â€” all without leaving the terminal.

What can you do with it?

CommandPurpose
authStore, rotate, and inspect API keys
initScaffold .prokodo/config.json in your project
verifyUpload project files and run a cloud verification
creditsCheck your credit balance
doctorDiagnose your environment in one command
